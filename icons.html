Icons

The server has had multiple icons throughout the months, most lasting for around a month before being updated or replaced completely.

James May

The first icon was the James May cheese thumbnail as this was popular with many members of the server at the time. Youtube video here

Altered UEA Logo

This was quickly changed to a photoshopped version of the official UEA icon with the words "Computer Science" added in to make the server look more useful for learning purposes and not just a meme server.

3D gif

It was discovered that the server logo could now be a gif so Arun asked the admins if anyone could create a logo in the form of a moving gif but no one did so Arun decided to make one himself in Unity. There were multiple iterations of this logo changing angles and lighting conditions to make the text more clear. It was based off the official UEA logo with the words "Computer Science" underneath.

Windows 98

When the online lectures started, one of the lecturers took a humerous comment seriously. The post was a question asking if they could use Windows 98 and the lecturer replied with "WINDOWS 98??!". This moment was deemed to be hilarious and became an inside meme so the server was renamed with reference to windows 98. When discord nitro users boosted the server, the banner was enabled and a Windows 98 logo was used. Later on, the main lecturers were photoshopped inside of the four Windows 98 logo quadrants which became the new logo.

Paper 98

A similar situation occured with the same lecturer but this time the question was along the line of "Can I use paper to do it?" and the lecturer, once again took the question seriously saying "PAPER??!", effectively replacing the old meme and so the server was renamed "UEA CS Paper". The icon was changed to a stack of paper and later on was made into a gif with the lecturer's faces in the Windows 98 logo printed on the paper. This remains the current logo today and is now the longest servering icon, not being changed after Arun graciously stepped down from being owner.

This was recreated for this website without a background and may be added to the server at some point.

Floppy 98

Arun had been playing around with new logo ideas for a few months before finally releasing the floppy disc logo on December the 8th aroun 3am.

These were animated originally in powerpoint with frame-by-frame pictures taken. To do the Christmas varient, a transparent version was first made and the background added outside of powerpoint.
